🪐 Misión 1: Llanura de Vectonia

Calibra tu RoverBot para recorrer una distancia exacta hasta el primer punto de control. Descubre la relación entre potencia del motor, tiempo de ejecución y distancia recorrida.

🎯 Objetivos:

• Comprender la relación potencia-distancia
• Realizar mediciones reales y registrar datos
• Construir tablas de datos y conclusiones

🛰️ Misión 2: Paso de Zycurva

Activa la conexión entre dos RoverBots para explorar nuevas zonas. Aprende cómo coordinar dos motores para crear movimientos de giro y avance rectilíneo.

🎯 Objetivos:

• Coordinar dos RoverBots para crear giros
• Descubrir relaciones entre tiempo y ángulos
• Trabajar en equipo para resolver desafíos

🔭 Misión 3: Cráter del Sensorium

Utiliza el sensor de distancia para que tu RoverBot se detenga automáticamente al detectar cráteres peligrosos. Explora el uso de condicionales en programación.

🎯 Objetivos:

• Usar condicionales simples (si... entonces...)
• Explorar el comportamiento del sensor de distancia
• Analizar márgenes de error en mediciones

🤝 Misión 4: Estación de Transferencia

Coordina dos equipos para transportar recursos energéticos. Planifica rutas complejas y sincroniza movimientos entre múltiples RoverBots.

🎯 Objetivos:

• Coordinar secuencias entre dos equipos
• Aplicar movimientos combinados complejos
• Resolver retos logísticos en equipo
